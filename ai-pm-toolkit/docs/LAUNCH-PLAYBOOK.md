## LinkedIn Launch Post (Draft)

```
I built an open-source toolkit for product managers using Claude.ai.

Here's the problem:
Most PMs prompt Claude from scratch every time. The output is generic. The formatting is inconsistent. You spend 15 minutes re-prompting to get something usable.

So I packaged everything I use daily into a public GitHub repo:

→ PRD generator (10-section structured output)
→ User story writer with Given/When/Then + Mixpanel events
→ Competitive analysis framework with scoring matrix
→ Release notes generator (internal + external variants)
→ KB article generator with step-by-step format
→ MCP workflow guides for Linear, Slack, and Notion

Every template is copy-paste ready. No setup. Works in Claude.ai free or Pro.

Why open source it?

Because "just prompt better" is not a strategy. PMs need reusable systems, not one-off prompts.

The repo is MIT licensed. Fork it. Adapt it. Contribute back.

Link in comments.

#ProductManagement #AI #Claude #OpenSource
```
